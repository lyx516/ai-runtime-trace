## 团队搭配经验
目标: 比较HNSW和IVF两种向量索引 | 团队: researcher, analyst, designer, critic, writer | 效果: 设计师与批评者形成有效的“对抗-细化”循环，通过反复质疑和调整假设，将模糊需求转化为可量化的决策依据，但缺少分析师和研究者导致数据支撑不足。
Gate建议: 建议在设计师首次输出后增加gate：若输出为重复提问或空方案，则自动触发批评者介入并标记为“需改进”，避免无意义循环。同时，在批评者提出关键约束后，应设置gate强制设计师更新假设条件后再进入下一轮评审。
