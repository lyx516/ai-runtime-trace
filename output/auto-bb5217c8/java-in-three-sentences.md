# Java 语言 · 三句话介绍

**Java** 是一种跨平台、面向对象的编程语言，采用"一次编写，到处运行"（Write Once, Run Anywhere）的理念，通过 Java 虚拟机（JVM）实现字节码与操作系统之间的解耦，从而在 Windows、Linux、macOS 等不同平台上无需修改即可运行。

**Java** 拥有自动内存管理（垃圾回收，GC）和强类型安全机制，内置丰富的标准库（Java Standard Edition），覆盖集合、并发、网络、IO、数据库访问等常见需求，使开发者能够专注于业务逻辑而非底层细节。

**Java** 广泛应用于企业级后端系统、Android 移动应用、大数据框架（如 Hadoop、Spark）和云计算服务，其成熟的生态系统、活跃的社区和长期的向后兼容性使其成为全球最受欢迎的编程语言之一。
